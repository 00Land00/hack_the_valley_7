# hack_the_valley_7
How to setup:
- install node js
- restart machine
- verify with "node -v" in terminal

How to run frontend:
- open terminal
- navigate to frontend folder
- run "npm start"
- to stop running, press "Ctrl + C"

How to run backend:
- open terminal
- navigate to backend folder
- run "npm start"
- to stop running, press "Ctrl + C"
